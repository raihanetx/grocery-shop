# **App Name**: Hello Canvas

## Core Features:

- Dynamic Greeting Display: Prominently displays a friendly, customizable greeting message on the page.
- Basic Visual Customizer: Provides simple UI controls to modify the text color and background color of the greeting display for personalization.
- Share Greeting: Allows users to easily copy the current customized greeting to their clipboard or generate a shareable link.

## Style Guidelines:

- The primary color is a calming blue-green: '#61BFD1'. This hue evokes serenity and clarity, fitting the welcoming and simple nature of the app.
- The background features a very light, desaturated blue-gray: '#F0F4F5'. It subtly hints at the primary color's hue while maintaining a clean, open, and inviting light aesthetic.
- An accent color, a vibrant and clean green: '#26D99C', provides clear contrast for interactive elements and highlights, bringing a fresh touch without being overly dominant.
- The font for all text is 'Inter' (sans-serif), chosen for its modern, clean, and highly readable qualities, which support a welcoming and user-friendly experience.
- The design emphasizes minimalism, employing ample whitespace around content, and centering the main greeting to draw focus and convey simplicity.
- Subtle, smooth fade-in and slide animations are used for greeting changes and customizer interactions, enhancing responsiveness without distraction.
